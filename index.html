Conversor de Medidas Web

Objetivo do Projeto

Desenvolver um site funcional que atue como um conversor de medidas web, permitindo aos usuários converterem medidas em pixels para vh, vw, e outras unidades. O projeto é o primeiro passo para a futura expansão em um site abrangente de Recursos para Interface do Usuário.

Requisitos Técnicos

Front-end:

  • HTML, CSS, Bootstrap para a estrutura e estilização responsiva.
  • Javascript para a interatividade e funcionalidades do conversor.
  • Interface intuitiva e amigável para o usuário.
  • Adaptação para diferentes tamanhos de tela e dispositivos.

Back-end:

  • A lógica do conversor será implementada em Javascript.

Distribuição de Tarefas

Protótipo

  • Criar a estrutura básica da página utilizando HTML.
  • Estilizar a página usando CSS e Bootstrap para garantir responsividade.

Javascript conversor e demais funções

  • Implementar a lógica do conversor em Javascript.
  • Garantir a interatividade e a precisão nas conversões.

Testes

  • Realizar testes de usabilidade e garantir que o conversor funcione corretamente em diferentes navegadores e dispositivos.

Cronograma

Reunião 1

  • Protótipo, HTML e estrutura inicial.
  • Início da implementação Javascript.

Reunião 2

  • Estilização com CSS e Bootstrap.
  • Continuação da implementação Javascript.

Reunião 3

  • Testes de usabilidade e ajustes.
  • Correção de bugs e melhorias.

Este documento é um guia inicial para o desenvolvimento do Conversor de Medidas Web. Qualquer ajuste ou modificação necessária ao longo do processo será discutido e implementado conforme a necessidade.
